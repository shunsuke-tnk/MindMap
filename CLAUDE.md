# CLAUDE.md

このファイルは Claude Code による開発のガイドラインです。

## プロジェクト概要

- **アプリ名**: MindMap
- **一言説明**: Webベースのリアルタイム共同編集マインドマップツール
- **背景・目的**: チームの会議やブレストで、リアルタイムにマインドマップを共同編集できるツールが必要。XMindのようなツールをWeb上で手軽に使えるようにする

## アプリのゴール

| 項目 | 内容 |
|------|------|
| **WHO**（誰のため） | チームメンバー（5〜10人規模） |
| **WHAT**（何を解決） | 会議・ブレストでアイデアをリアルタイムに共有・構造化したい |
| **HOW**（どうやって） | URL共有だけで複数人が同時にマインドマップを編集できるWebアプリ |

## ターゲットユーザー

- **メインユーザー**: 特定のチーム・仲間（5〜10人）
- **利用シーン**: 会議・ブレストでリアルタイムにアイデアを出し合う
- **ユーザーの課題**: アイデアの共有・構造化を同時に行うツールがない
- **対応デバイス**: PC中心（マウス操作・キーボードショートカット重視）

## 主要機能

### MVP（最初のリリース）

| # | 機能名 | 説明 | 優先度 |
|---|--------|------|--------|
| 1 | マインドマップ作成・編集 | ノードの追加・削除・テキスト編集、階層構造の構築 | 必須 |
| 2 | リアルタイム共同編集 | URL共有で複数人が同時にマップを編集、カーソル共有 | 必須 |
| 3 | マップの保存・読み込み | 作成したマップを自動保存し、後から再開できる | 必須 |

## 画面構成

| # | 画面名 | 概要 | 主な操作 |
|---|--------|------|----------|
| 1 | トップページ | アプリの入口。新しいマップを作成する | 「新しいマップを作る」ボタン |
| 2 | マップ編集ページ | マインドマップの作成・編集・共同作業 | ノード操作、レイアウト切替、共有URLコピー |

## UI/UX方針

- **デザインテイスト**: カラフル・ポップ（XMind風、階層ごとに色分け）
- **レスポンシブ**: PCファースト（タブレットでも崩れない程度に対応）
- **参考**: XMind
- **基本方針**:
  - ノードは角丸デザインで階層ごとに色分け
  - ダブルクリックでインライン編集
  - キーボードショートカット: Tab（子ノード追加）、Enter（兄弟ノード追加）、Delete（削除）

## 技術仕様

- **フレームワーク**: Next.js 15（App Router）
- **スタイリング**: Tailwind CSS
- **言語**: TypeScript（strict mode）
- **ノード描画**: @xyflow/react（React Flow v12）
- **レイアウト計算**: d3-hierarchy + d3-flextree
- **リアルタイム同期**: Liveblocks（CRDT、プレゼンス、永続化）
- **パッケージマネージャ**: npm

### 環境変数

```
NEXT_PUBLIC_LIVEBLOCKS_PUBLIC_KEY=   # Liveblocks パブリックキー（クライアント用）
LIVEBLOCKS_SECRET_KEY=               # Liveblocks シークレットキー（サーバー用）
```

### コマンド

```bash
npm install          # 依存パッケージのインストール
npm run dev          # 開発サーバー起動（http://localhost:3000）
npm run build        # 本番ビルド
npm run start        # 本番サーバー起動
npx tsc --noEmit     # 型チェック
```
